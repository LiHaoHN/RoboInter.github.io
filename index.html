RoboInter: Interaction-Centric Spatial Annotation for Robotic Manipulation

A comprehensive framework for annotating fine-grained spatial interactions in robotic manipulation datasets

Anonymous Authors

Paper Code RoboInter-Data RoboInter-VQA

Abstract

Understanding and predicting spatial interactions between robots, objects, and environments is crucial for robotic manipulation. We introduce RoboInter, an interaction-centric annotation framework that provides comprehensive spatial annotations for robotic manipulation datasets. Our framework captures fine-grained details including object bounding boxes, gripper states, contact points, trajectory predictions, affordance regions, and placement proposals.


Built upon real-world datasets including DROID and RH20T, RoboInter provides rich annotations that enable training vision-language models for robotic understanding and generation tasks. We also release RoboInter-VQA, a visual question answering benchmark for evaluating spatial reasoning capabilities in robotic contexts.
